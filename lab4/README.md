# Лабораторная работа №4: Пул потоков (POSIX Threads)

## Что реализовано

- Пул из `3` рабочих потоков
- Очередь задач фиксированного размера `10`
- Отправка задач через `pool_submit()`
- Защита очереди через `pthread_mutex`
- Уведомление о новых задачах через `sem_t`
- Корректное завершение через `pool_shutdown()`

## Файлы

- `threadpool.h` — API пула
- `threadpool.c` — реализация пула
- `client.c` — простой клиентский пример
- `Термины_и_понятия_к_лаб4.md` — ключевые термины по теме

## Сборка и запуск (Linux/macOS с gcc)

```bash
make
./example
```

## Очистка

```bash
make clean
```
